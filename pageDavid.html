GitHup Learning Lab

I enrolled in the GitHub Learning Lab, signed up for, and completed, the first course, Introduction to GitHub. It was helpful, without doubt, but rather poorly planned. If you're thinking of trying it yourself, make sure you make a copy of this URL, https://lab.github.com/githubtraining/introduction-to-github.

Believe me, you'll be glad you did, because the course leaves you several times floundering in a limbo land, not knowing what to do next. Following this link will 'save your bacon'.
